
# Прогнозирование температуры масла трансформатора

##  Описание проекта

Проект посвящен прогнозированию температуры масла трансформатора на основе временных рядов. Цель — сравнить эффективность статистических и машинных методов прогнозирования для обеспечения надежной работы энергетического оборудования.

**Датасет:** Oil Temperature Transformer Dataset 
**Горизонт прогноза:** 7 дней  
**Частота данных:** Ежедневные наблюдения (Daily)

## 🔍 Разведочный анализ данных (EDA)

### Основные характеристики
- **Период:** 2016-07-01 - 2018-06-26 
- **Пропуски:** 0
- **Средняя температура:** 13,32°C
- **Стандартное отклонение:** 8,57°C

### Визуализации
1. **Временной ряд температуры** — явно выражена сезонность
2. **Распределение температуры** — близко к нормальному с небольшим правосторонним скосом
3. **Boxplot по месяцам** — зимой температура ниже, летом выше
4. **Автокорреляция (ACF/PACF)** — значимая корреляция на лагах 7, 14, 21, 28 дней

### Ключевые выводы EDA
-  Ярко выраженный **годовой сезонный паттерн**
-  **Нестационарность** ряда — требуется дифференцирование
-  Сильная автокорреляция — подходит для ARIMA-моделей
-  Отсутствие аномальных выбросов

## 📈 Статистические модели (Задание №2)

### Метрики качества

| Модель | MAE (°C) | RMSE (°C) | Improvement vs Baseline |
|--------|----------|-----------|-------------------------|
| **SeasonalNaive** | 1.456 | 1.789 | - |
| **ARIMA** | 0.870 | 1.200 | 65.6% |
| **AutoARIMA** | 1.66 | 2.32 | 34.7% |

### Лучшая статистическая модель
**ARIMA(2,0,2)** — MAE = **0.870°C**

**Параметры:**
- p=2 (авторегрессия)
- d=0 (дифференцирование для стационарности)
- q=2 (скользящее среднее)

**Почему лучше AutoARIMA?**  
Ручной подбор параметров позволил найти оптимальную конфигурацию, которую AutoARIMA пропустил из-за ограничений автоматического алгоритма.
